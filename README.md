Literalura
Literalura es una aplicación de consola desarrollada con Java y Spring Boot que permite buscar libros utilizando la API pública de Gutendex y guardar la información en una base de datos.

Funcionalidades

La aplicación permite:
1- Buscar libros por título utilizando una API externa.
2- Guardar libros en una base de datos.
3- Listar todos los libros registrados.
4- Listar autores registrados.
5- Buscar autores que estaban vivos en un año determinado.
6 -Buscar libros por idioma.

Tecnologías utilizadas

Java
Spring Boot
Spring Data JPA
PostgreSQL
API Gutendex

¿Cómo ejecutar el proyecto?

Clonar el repositorio.
Configurar la base de datos en el archivo:
src/main/resources/application.properties
Ejecutar la aplicación:
LiteraluraApplication.java
Usar el menú de la consola para interactuar con el sistema.
